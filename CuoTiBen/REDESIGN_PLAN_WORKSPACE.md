# 学习工作台 UI 重设计方案

## 设计愿景

将"错题本"从"功能拼接的学习 App"转变为"专业的学习工作空间"，参考 Notion、Obsidian 等桌面级应用的 UX 理念，打造 iPad 优先、iPhone 适配的现代化学习工具。

---

## 一、设计原则

### 1.1 视觉风格
- **温和玻璃感**: 轻盈的磨砂效果，非厚重的毛玻璃
- **低饱和蓝灰基调**: 主色调采用温和的蓝灰色系，避免纯黑白
- **克制使用强调色**: 仅在关键操作、知识点标记处使用高饱和色彩
- **明确的层级关系**: 通过阴影、透明度、间距建立清晰的视觉层次

### 1.2 交互理念
- **微妙的动效**: 200-300ms 的平滑过渡，避免夸张动画
- **明确的反馈**: 点击压缩、悬停高亮、选中状态清晰
- **减少干扰**: 无意义的闪烁、渐变、跳动一律去除
- **符合直觉**: 操作逻辑遵循 iOS 人机界面指南

### 1.3 布局策略
- **iPad 优先**: 充分利用大屏优势，采用双栏、浮动面板等桌面级布局
- **iPhone 适配**: 保持核心功能可用，简化复杂交互
- **呼吸感**: 增加留白，减少信息密度，提升可读性
- **一致性**: 统一组件语言，跨页面保持一致的视觉体验

## 二、设计系统 Token

### 2.1 颜色系统

#### 基础中性色 (Base Neutrals)
```swift
backgroundPrimary:     #F6F7F9  // 主背景：温和浅蓝灰
backgroundSecondary:   #F0F1F5  // 次要背景：略深蓝灰
cardBackground:        #FFFFFF  // 卡片背景：92% 不透明白色
glassSurface:          #FFFFFF  // 玻璃表面：75% 透明度
```

#### 文本颜色 (Text Colors)
```swift
textPrimary:           #2E333D  // 主标题：深灰蓝
textSecondary:         #525760  // 正文：中等灰蓝
textTertiary:          #7A7F8A  // 辅助文字：浅灰蓝
textPlaceholder:       #A3A8B2  // 占位文字：更浅的灰
```

#### 强调色 (Accent Colors) - 克制使用
```swift
accentIndigo:          #475C9E  // 主强调：靛蓝色 (关键操作)
accentTeal:            #38858F  // 次要强调：青色 (知识点)
accentCoral:           #EB6B66  // 提醒：珊瑚色 (非正红)
accentTurquoise:       #3DA48F  // 成功：松石绿
accentAmber:           #EBAD47  // 警告：琥珀色
```

#### 语义色 (Semantic Colors)
```swift
quoteBackground:       #EBEEF7  // 引用块背景：极淡靛蓝
quoteBorder:           #475C9E  // 引用块边框：40% 透明度
knowledgePointBg:      #E6F4F4  // 知识点芯片：淡青色
knowledgePointBorder:  #38858F  // 知识点边框：50% 透明度
```

### 2.2 字体排印 (Typography)

#### 字体系列
- **主字体**: SF Pro Rounded (圆角，亲和力)
- **等宽字体**: SF Mono (代码、引用)

#### 字号层级
```
Display Large:   34pt Bold      // 页面级超大标题
Display Medium:  28pt Bold      // 主要页面标题
Display Small:   24pt Bold      // 次级页面标题

Headline Large:  22pt Semibold  // 大区块标题
Headline Medium: 20pt Semibold  // 标准区块标题
Headline Small:  18pt Semibold  // 小子标题

Body Large:      17pt Regular   // 大正文 (行距 6pt)
Body:            15pt Regular   // 标准正文 (行距 5pt)
Body Small:      14pt Regular   // 小正文 (行距 4pt)

Label Large:     14pt Medium    // 大标签
Label:           13pt Medium    // 标准标签
Label Small:     12pt Medium    // 小标签

Caption:         12pt Regular   // 图注/脚注
Caption Small:   11pt Regular   // 小图注
```

### 2.3 间距系统 (Spacing)

基于 4pt 网格:
```
xs:   4pt   // 极小间距
sm:   8pt   // 小间距
md:   12pt  // 中小间距
lg:   16pt  // 标准间距
xl:   20pt  // 大间距
xxl:  24pt  // 超大间距
xxxl: 32pt  // 极大间距
```

**页面边距**:
- iPad: 32pt
- iPhone: 20pt

**卡片内边距**: 20pt
**面板内边距**: 24pt

### 2.4 圆角系统 (Corner Radius)

```
sm:      6pt   // 芯片、小按钮
md:      10pt  // 按钮、输入框
lg:      14pt  // 卡片
xl:      20pt  // 大卡片、面板
xxl:     28pt  // 大型容器
capsule: 999pt // 胶囊形
```

### 2.5 阴影系统 (Shadows)

```swift
// 悬浮卡片
cardFloating: color=#99A3B3 opacity=12%, radius=12, x=0, y=4

// 悬浮面板
panelFloating: color=#99A3B3 opacity=18%, radius=16, x=0, y=6

// 悬浮按钮
buttonFloating: color=#475C9E opacity=30%, radius=10, x=0, y=4

// 按下状态
pressed: color=#99A3B3 opacity=12%, radius=6, x=0, y=2

// 模态层
modal: color=#99A3B3 opacity=28%, radius=24, x=0, y=12
```

### 2.6 动画系统 (Animations)

#### 时长
```
quick:     100ms  // 快速反馈
standard:  200ms  // 标准过渡
smooth:    300ms  // 平滑过渡
elegant:   400ms  // 优雅过渡
slow:      500ms  // 大尺度变化
```

#### 弹簧预设
```swift
springLight:   response=0.15, dampingFraction=0.85  // 小按钮、芯片
springStandard: response=0.25, dampingFraction=0.82 // 卡片、面板
springSoft:    response=0.35, dampingFraction=0.80  // 页面过渡
springBouncy:  response=0.30, dampingFraction=0.70  // 强调反馈
```

#### 转场预设
```swift
fadeInSlideUp: 淡入 + 上移 (内容切换)
scaleFadeIn:   缩放 + 淡入 (卡片展开)
slideIn:       滑入 + 淡入 (侧边面板)
```

## 三、核心组件库

### 3.1 基础组件

#### AppPageHeader
**用途**: 页面头部，包含标题、副标题、操作按钮

**结构**:
```
┌─────────────────────────────────────┐
│ ← [返回]  页面标题      [设置] ⚙️  │
│           今日学习概览               │
└─────────────────────────────────────┘
```

**特性**:
- 可选的前导操作 (返回按钮)
- 主标题 + 副标题双层结构
-  trailing actions 支持多个操作

---

#### SectionHeader
**用途**: 区块头部，图标 + 标题 + 可选操作

**结构**:
```
┌─────────────────────────────────────┐
│ 📊  学习进度            [查看全部]  │
├─────────────────────────────────────┤
```

**特性**:
- SF Symbols 图标 + 浅色背景
- 右侧可选操作按钮
- 底部分割线 (可选)

---

#### SegmentedSwitch
**用途**: 分段切换器，无闪烁平滑过渡

**结构**:
```
┌──────────┬──────────┬──────────┐
│  全部    │ 进行中   │  已完成  │
└──────────┴──────────┴──────────┘
```

**特性**:
- 支持图标 + 文字
- 紧凑/标准两种模式
- 选中状态带阴影
- 无闪烁动画

## 五、iPad vs iPhone 差异化布局

### 5.1 iPad 布局特点

#### 优势利用
- **双栏布局**: 列表 + 详情同时显示
- **浮动面板**: 结构树、大纲可悬浮拖动
- **宽屏阅读**: PDF 占据 60% 宽度，侧边栏 40%
- **多任务**: 支持 Slide Over、Split View

#### 具体尺寸
```
笔记列表宽度：380pt (固定)
笔记详情宽度：≥500pt (自适应剩余空间)
浮动面板宽度：≤400pt (可调整)
页面边距：32pt
卡片最大宽度：720pt (可读性优化)
```

### 5.2 iPhone 布局特点

#### 简化策略
- **单栏布局**: 列表/详情分页显示
- **全屏模态**: 工作台、编辑器全屏展开
- **底部导航**: Tab Bar 切换主要功能
- **紧凑间距**: 减少留白，提升信息密度

#### 具体尺寸
```
页面边距：20pt
卡片水平间距：20pt
列表项高度：≥60pt (易点击)
按钮最小高度：44pt (无障碍)
```

### 5.3 响应式断点

```swift
compact:   < 600pt   // iPhone, iPhone Plus
regular:   600-900pt // iPad mini, iPad (横屏)
expanded:  > 900pt   // iPad Pro, 大屏设备
```

## 六、交互模式规范

### 6.1 卡片压缩反馈

**触发**: 用户点击卡片
**动画**:
```swift
withAnimation(.spring(response: 0.25, dampingFraction: 0.82)) {
  scale = 0.98
}
// 100ms 后恢复
DispatchQueue.main.asyncAfter(deadline: .now() + 0.1) {
  withAnimation(.spring(response: 0.25, dampingFraction: 0.82)) {
    scale = 1.0
  }
}
```

### 6.2 内容切换过渡

**场景**: Tab 切换、列表刷新
**动画**:
```swift
.transition(.asymmetric(
  insertion: .move(edge: .bottom).combined(with: .opacity),
  removal: .opacity
))
```

### 6.3 目标高亮

**场景**: 点击目录跳转、知识点链接
**效果**:
- 目标卡片边框高亮 (靛蓝色，2pt)
- 背景色短暂闪烁 (5% 透明度)
- 持续时间 1 秒

### 6.4 芯片反馈

**场景**: 点击知识点芯片
**动画**:
```swift
// 压缩
withAnimation(.spring(response: 0.15, dampingFraction: 0.85)) {
  isPressed = true
}
// 恢复
DispatchQueue.main.asyncAfter(deadline: .now() + 0.1) {
  withAnimation(.spring(response: 0.15, dampingFraction: 0.85)) {
    isPressed = false
  }
}
```

### 6.5 分段切换无闪烁

**关键**: 使用 `.animation(nil, value: isSelected)` 避免闪烁
```swift
Button(action: {
  withAnimation(.spring(response: 0.15, dampingFraction: 0.85)) {
    selection = newValue
  }
}) {
  // 按钮内容
}
.animation(nil, value: isSelected) // 关键！
```
